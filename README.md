Description:
This project is a small, state‑driven UI library inspired by the core ideas behind React and other modern front‑end frameworks. It gives you a minimal but functional reactive system where UI updates automatically re-render when the internal state changes.
The library includes a simple subscription mechanism, a state container, and a very small rendering pattern that shows how frameworks handle UI updates under the hood. It’s perfect for developers who want to understand reactivity without the complexity of a large framework.
Highlights:

A tiny but functional reactive state system
Demonstrates the basic principles behind React/Vue/Svelte
Automatically updates the DOM when the state changes
Clean and easy to read code
Great educational tool or portfolio project
Works in any browser with no external dependencies
